String and NSString

The String struct in Swift has a counterpart in Objective-C, the NSString class. NSString is a very mature and rich class stretching back to the beginning of NeXT. Instead of re-writing all of it's optimized goodness, the Swift String can just use all the functions in NSString.

Let's look at the documentation for NSString. In Xcode, go to the Help➙Documentation and API Reference menu item. Enter "NSString" in the top search bar.

That's a lot of functions.
