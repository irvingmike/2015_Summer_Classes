Calling an Objective-C Function

The count() function above is part of Swift and is written in Swift.

Next, we'll call a function that is written in Objective-C.

This function will break up a string around the spaces. Here's how we would call the function in Objective-C.

1
NSArray *words = [sample componentsSeparatedByString: @" "];

That may look a little strange to a new programmer who hasn't encountered it before. Swift can call this same function using the syntax we've been learning.

1
let words = sample.componentsSeparatedByString(" ")
