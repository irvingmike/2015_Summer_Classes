Case Insensitive Compare

What if we don't care about the case of the strings we are comparing? One way to do this is to look at the companion class to Swift's String: NSString. This is the Objective-C class that can be used with String.

Open the documentation window in Xcode and search for NSString. Let's review this together.

OK, we're back. We found this Obj-C method which will work for us.

1
func caseInsensitiveCompare(aString: String) -> NSComparisonResult

We understand that this will compare one string to another, but what is NSComparisonResult?

NSComparisonResult is an Obj-C enum. Here are the values in Swift.
