An Initializer is a special function named simply init like this.

1
2
3
    init() {               

    }

There are no words in front like func or public, it's just init.

When you use the code Pet() you are actually calling the init method. Let's see that in action. Add the init method to the class with a println() statement in it so we can tell when it gets called. Add this code right after the variable.

1
2
3
    init() {
        println("In the init() method!")
    }

Open the console if you haven't already. As soon as you add the code you should see this.

1
2
In the init() method!
In the init() method!

OK, we can see that they run but what are they for?

They are for initializing an instance, just like their name. We put code in them to set up our object. Let's add default values for our variables to see how this works.

1
2
3
4
5
6
7
    init() {
        println("In the init() method!")
        name = "<name>"
        species = "<unknown>"
        weight = 0.0
        height = 0.0
    }
