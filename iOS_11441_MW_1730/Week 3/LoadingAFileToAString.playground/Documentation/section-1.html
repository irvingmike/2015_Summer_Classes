Open a terminal window and run this command.

1
$ cd Documents/Shared\ Playground\ Data/

If you see this message then you don't have that directory. 

1
-bash: cd: Documents/Shared Playground Data/: No such file or directory

If you don't have that directory run this command. This will create a directory that Swift playgrounds can access. 

1
mkdir Documents/Shared\ Playground\ Data

Next, run this command.

1
cp /usr/share/dict/propernames .

Now we have a nice file in our shared playground directory that we can use in this playground.

We want to load that file into a string so we can use it. Here's how. (Change user to your account name.)
