Computed Properties

We can also have properties that use stored properties to computed or create a value. They are called computed properties. They start out looking like other properties with a var. But, then...

    var point: (Double, Double) {
        get {
            return (pointX, pointY)
        }
        set {
            pointX = newValue.0
            pointY = newValue.1
        }
    }

Whoa! That looks like a cross between a variable and a function. And that's basically what it is.

newValue is an automatic parameter that Swift creates for you. Add this property to your class.
