Swift variables and constants can't be nil (or null from other languages). This has some serious benefits for modern programming. For one, this allows the compiler to apply some serious optimizations. Swift code has very high performance. It also eliminates a whole category of errors.

Numbers in Swift

Swift has numbers that represent integer values and floating point numbers.

Integers

Integers in Swift come in sizes that are 8, 16, 32, and 64 bits. Most of the time we don't need to think about this and can use the simple form.

  • Signed integers
    • Int – This is the form to use most of the time and its size will vary depending on the OS.
    • Int8
    • Int16
    • Int32
    • Int64
  • Unsigned integers – We will want to default to using an Int instead of these.
    • UInt
    • UInt8
    • UInt16
    • UInt32
    • UInt64

We can determine the maximum and minimum values of each of these with the .min and .max properties.
