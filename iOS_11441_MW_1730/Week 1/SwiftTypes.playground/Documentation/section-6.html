Booleans in Swift

Swift has a dedicated type for a boolean value that can only be true or false. The type is named Bool and you code it like this.

1
2
3
let tooHot = true
let tooCold = false
let justRight = false

They can be used in if statements like this

1
2
3
4
5
6
7
8
9
if tooHot {
    println("That's too hot.")
} else if tooCold {
    println("That's too cold.")
} else if justRight {
    println("Ah, that's just right.")
} else {
    println("That's weird.")
}
