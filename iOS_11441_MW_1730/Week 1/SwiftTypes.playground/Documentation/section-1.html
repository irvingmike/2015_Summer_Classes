Swift is a language with strict typing. That means a variable is one type for the life of an application. It can't be changed to a different type like in JavaScript. Once a String always a String!

When we have code like this how does Swift know what types they are? Swift uses something called Type Inference. The type of the variable or constant is inferred from the value you assign it.

1
2
let minimumHeightInMeters = 1.2192
let name = "Fred Flintstone"

So, name becomes a String. How about minimumHeightInMeters? It could be a Double or a Float. We can tell by Control-clicking on the variable. Go ahead and enter those variables and see what Xcode says.
