What if you wanted it to be a Float for some reason? Are we out of luck? Nope.

Type Annotation

We can annotate a variable or constant by adding the type. We do it like this:

1
2
let minimumHeightInMeters: Float = 1.2192
let name: String = "Fred Flintstone"

Type Annotations are not required. We use them when we need to for clarity or when we need things to be a certain type.

What's with this?

We are used to declaring variables without giving them an initial value. That's normal and expected. Here's a sample in Java.

2
3
4
// Java:

private int counter;
private String firstName;
