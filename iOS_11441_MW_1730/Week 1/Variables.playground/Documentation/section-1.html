Simple Variables

The simplest variable begins with the word var. Then comes a name, like count. Then an equal sign = followed by a value. Like this:

1
var count = 10

If you want a number, type a number by itself. If you want a string then use double quotes.

1
var city = "Madison"

There's no semicolon at the end of the statements. Yeah!
