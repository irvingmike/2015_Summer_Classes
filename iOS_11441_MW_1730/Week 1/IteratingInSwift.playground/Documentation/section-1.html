This should be very familiar to everyone. Lots of programming is about doing something repeatedly and very fast. There are many ways to do that and we can do most of them in Swift. We'll start with the main looping mechanisms we're used to, for and while.

For Loops

There are two types of for loops in Swift.

  • for-condition-increment – The classic that is in almost all languages.
  • for-in – The Swift form of what some know as a for-each loop.

for-condition-increment

This classic loop traces it's roots back to the beginning of modern languages. Here's the form.

1
2
3
for var index = 0; index < 5; index++ {
    println("index: \(index)")
}
