Looping Through Arrays

You should already know some about arrays and looping so let's combine them. Looping through an array can be very simple. Here's the basic form. I've included the array of names we used before.

1
2
3
4
5
var names = ["Bob", "Dakota", "Joe", "Ellen", "Josie", "Tim", "Gene", "Phil", "Suzie", "Bill"]

for name in names {
    println("Hello, \(name)")
}

How does it work?

It loops the right number of times automatically! With each loop the next element is placed into the constant name. You can make that constant any name you'd like that makes sense.

Why is name a constant? Shouldn't it be a var? Good question! Swift uses a constant by default and sets the next element into it at the top of each loop. It's a constant because we can't change its value inside the loop.
