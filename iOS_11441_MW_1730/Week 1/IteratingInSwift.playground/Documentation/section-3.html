Ranges

  • Closed range operator: ...
    • Inclusive – The values on either end are included in the range.
  • Half-open range operator: ..<
    • Half-exclusive – The value on the left in included, the value on the right is excluded.

for-in With No index

If you don't need the index value when you are looping there is no need to create it. We use and underscore _ instead of an index.

1
2
3
for _ in 1...5 {
    println("In the loop doing something important!")
}
