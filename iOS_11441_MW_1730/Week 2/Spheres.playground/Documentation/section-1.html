Surface Area

In the following code block, write a function that calculates the surface area of a sphere. You will need to use M_PI for Pi. The pow() function might be handy, too. The function will have one parameter that is a Double. A good name might be sphereSurfaceAreaWithRadius(). It will return a Double that is the calculated surface area.
