Strings and Numbers

Frequently, we want to make a string that contains text and a number.

The temperature is 85 degrees.

What if the 85 is in a variable and we need to concatenation it to the text? Our first guess might be to do this:

1
2
3
// Bad Code!
var temperature = 85
var currentTemperature = "The temperature is " + temperature + " degrees."

However, we will immediately see something to the left of the line:

This means the line has an error. We can see what the error is by clicking on the little red icon.

In this case the error is telling us that we can't concatenate strings and numbers. Since that is something we need to do often there must be another way. In Swift this is called String Interpolation.

String Interpolation

The + operator only works on strings. We need to be able to create a string with other types, too. You've probably seen that in other languages. Here's how Swift does it. 

1
2
var temperature = 85
var currentTemperature = "The temperature is \(temperature) degrees."

We first make a string with a set of double quotes. Then add a back-slash and a set of parenthesis.

  • "text can be on \(<data or variables here>) either side"
