Combining Strings in Swift

When we combine strings together in a programming language it's called concatenation. Let's learn a couple of ways of doing this. The first way is using the concatenation operator, which is the plus sign: +. We can add two strings:

1
var text = "One string, " + "two strings!"

The result will be: One string, two strings!

And we can also use the concatenation operator with variables. 

1
2
var stringOne = "This is the first string. "
var stringTwo = stringOne + "This is another string."

The resulting value in stringTwo will be:

This is the first string. This is another string.
