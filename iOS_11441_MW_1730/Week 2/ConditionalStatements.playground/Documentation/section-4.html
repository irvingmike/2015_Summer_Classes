Displaying a Bool.

Swift shares another feature of JS. If you display a Bool it will show true or false. Try this, output a few statements like these. (But, not these.) Wherever it says true or false you need to have a \(...) with a comparison inside.

  • 5 < 6 is true
  • 100.0 == (10 * 10) is true
  • 10 == 10.1: false
  • 10 is not greater to or equal to 100: true
