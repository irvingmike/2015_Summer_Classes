Adding a Parameter

When we call a function we can send some data to it. You've done this before. Here's where Swift starts showing its differences from most other languages. There are many ways to create parameters. We just have to start in on learning them. The first thing to see is that Swift parameters are named. They have a name and a type. The name comes first like this.

  • person: String
    • person is the name of the parameter.
    • String is the type of the parameter.
    • The : is required.

Here's a full function with one parameter.

2
3
func sayGreetingToPerson(person: String) {
    println("Hello, \(person)!")
}
