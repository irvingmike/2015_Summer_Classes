The type of Swift arrays is a set of square brackets with the type of the elements inside. Here's the type for an array of Strings.

  • [String]

An array of integer numbers would have this type.

  • [Int]

If our array is a variable, declared with a var, then we can add things to it. An array can also be a constant. If declared like this we can't change the array.

1
let counts = [5, 10, 74, 22, 123, 654, 999]

Accessing Array Elements

To retrieve one of the elements in an array we use this syntax.

1
let oneName = names[4]
