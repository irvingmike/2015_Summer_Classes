Making Empty Arrays in Swift

We need the ! after last because the array could be empty. In that case there is no last element. But, names.last! will fail if it's empty. We need an easy way to test if an array is not empty.

  • names.isEmpty

The isEmpty property is part of a Swift Array. However, Swift also lets us use the array that's written in Objective-C called NSArray. If we try to make an empty array like this it makes the wrong one.

1
2
// NSArray, not a Swift Array!
var names = []

If we make one like this it makes a Swift Array

1
2
// Swift!
var names: [String] = []

Then we can test the array to make sure it's empty before we do anything with it.

1
2
3
4
5
if !names.isEmpty {
    println(names.last)
} else {
    println("names is empty")
}

What happened with the first empty array? Even Swift arrays have to have types. With var names = [] there are no types. Swift just makes an array that doesn't have types, an NSArray.
