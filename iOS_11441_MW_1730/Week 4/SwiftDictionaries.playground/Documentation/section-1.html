A Dictionary in Swift

In programming, a dictionary is this same concept brought into code. Imagine a class where you could enter a word and it would return the definition. A Swift Dictionary does that and more.

Here's how we would define a Swift dictionary that would hold a dictionary of words.

1
var wordDictionary: Dictionary<String, String>

We can also also use this shortcut.

1
var wordDictionary: [String : String]

Terms

Notice that we use these terms.

  • Key – This is the general concept of the word we use to lookup the definition in a paper dictionary.
  • Value – This is a general term for what we get when we use a key to lookup something. It could be text, or images, or sounds, or instances of classes, or even other collections like an array.

Dictionary Example

Let's make a dictionary of words and definitions as an example.

1. Declare the dictionary variable.

1
var wordDictionary: Dictionary<String, String>

2. Create the instance of the dictionary itself. The [:] symbol is like the [] symbol for creating an array. 

1
wordDictionary = [:]

3. Add some dictionary entries. Here's the syntax for one entry.

1
wordDictionary["car"] = "a road vehicle, typically with four wheels, powered by an internal combustion engine and able to carry a small number of people"
