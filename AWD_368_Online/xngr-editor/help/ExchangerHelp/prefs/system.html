System Preferences

Figure 105. System Preferences

The Host Address and Port Number can be set to configure proxy access to remote resources.

The System tab facilitates adding Jars for use by XSLT transformations that rely on extensions. Alternatively, from the command line, use java -classpath /test/myextension.jar -jar lib/xngr-loader.jar for example.

Changing the Look and Feel for the application requires a restart to take effect.

The System Preferences tab no longer allows the user to control initial and maximum memory used by the application when starting up. Instead, from the command line, use java -ms64M -mx128M -jar lib/xngr-loader.jar, for example, or modify the appropriate entries in the ".lax" file in the installation directory.
