SVG

Exchanger XML Editor has built-in support for viewing, editing and converting Scalable Vector Graphics (SVG) content. Open the file moonPhases.svg from the SVG project in the editor. To view, select Tools->Show SVG and the graphic is opened in the SVG Viewer. The viewer supports zooming in/out, panning, rotating, printing, etc.

Figure 1. Viewing SVG

To convert the graphic to other formats such as JPEG, PNG, and TIFF use Tools->Convert SVG.
