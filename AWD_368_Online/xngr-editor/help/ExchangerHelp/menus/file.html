File Menu

New Create a new document in the editor. Ctrl+N
Open... Opens an existing file in the editor. Ctrl+O
Open Remote... Opens a remote file
Close Closes the active document. If there are unsaved changes, will prompt the user to save the document. Ctrl+W
Close All Closes all documents. If there are unsaved changes, will prompt the user to save each document. Ctrl+Shift+W
Reload Reloads the current document. If there are unsaved changes, will prompt the user to save the document.
Save Saves the active document to disk using the document's current file name. Ctrl+S
Save As... Saves the active document with a name you provide using a file chooser dialog box.
Save As Remote Saves the active document with a name you provide as a remote file.
Save All Saves all documents to disk using the document's current file name. Ctrl+Shift+W
Import Import Data from various sources.
    From Text File...     Import Data from structured text files
    From Excel File...     Import Data from Excel spreadsheets
    From Database Table...     Import Data from a relational database table
    From SQL/XML Query...     Import Data using an SQL/XML Query
Save As Template... Save the current document so that it can be used as a template when creating a new document with File->New.
Manage Templates Create, update or delete templates.
Page Setup Displays the Page Setup dialog that controls print settings such as page size, orientation and margins.
Print Prints the current document, using the settings in the Preferences Print tab. Ctrl+S
Preferences Displays the Preferences tabs, used for controlling editing, viewing, printing, and network settings.
Exit Shuts down the Application. If there are unsaved changes, will prompt the user to save the document(s).
