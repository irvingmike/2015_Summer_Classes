Transform Menu

Execute XSLT... If the active document is an XSLT file, Transform->Execute XSLT will prompt for an input XML file. If the active document is an XML file, then Transform->Execute XSLT will prompt for an XSLT file. The output of the transformation can be sent to a file, to a new document, to the console (i.e. the Output tab at the bottom of the screen) or can be opened using the default Internet browser.
Execute Last XSLT Repeat the previous XSLT transformation. Having this on the menu facilitates the creation of a keyboard shortcut which allows transformations to be easily re-run.
Execute FO... If the active document is an FO file, Transform->Execute FO will prompt for an input XML file. If the active document is an XML file, then Transform->Execute FO will prompt for an FO file. The output of the transformation can be sent to a built-in viewer or saved to file in a number of different formats including PDF, PostScript, text and SVG.
Execute Last FO Repeat the previous FO transformation. Having this on the menu facilitates the creation of a keyboard shortcut which allows transformations to be easily re-run.
Execute XQuery... Transform->Execute XQuery will prompt for an input XML document (or you can use the current document) and for a query document and will direct the result to a file, to a new document, to the console (i.e. the Output tab at the bottom of the screen) or to the default Internet browser.
Execute Last XQuery Repeat the previous XQuery. Having this on the menu facilitates the creation of a keyboard shortcut which allows queries to be easily re-run.
Execute Scenario... Displays a list of Scenarios, from which the user can choose one to execute
Execute Last Scenario Repeat the previous Scenario. Having this on the menu facilitates the creation of a keyboard shortcut which allows scenarios to be easily re-run.
XSLT Debugger For integrated or stand-alone debugging of style sheets. Displays multiple style sheet files and multiple input files above one or more output files that are generated as a result of the transformation.
Manage Scenarios Allows a user to create, update or delete scenarios
