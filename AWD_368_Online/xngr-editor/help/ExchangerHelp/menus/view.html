View Menu

Schema Switches the current view to Schema Viewer mode for browsing the Schema associated with the active document. Only available if a schema has been opened or if a Type has been set for the active document.
Outliner Switches the current view to Outliner mode for editing the active document without the need for tags. Only available if a schema has been opened or if a Type has been set for the active document.
Editor Switches the current view to Editor mode.
Viewer Switches the current view to Viewer mode for browsing and searching the active document.
Grid Switches the current view to Grid mode for tag-free editing.
Highlight Highlights elements and attributes from different namespaces with different colors.
Collapse All Collapses the tree view when in Viewer, Outliner or Schema Viewer mode.
Expand All Expands the tree view when in Viewer, Outliner or Schema Viewer mode.
Synchronise Selection Controls cursor synchronisation between the Editor and Viewer modes.
Toolbars Controls the visibility of the different toolbars
    Standard Buttons Controls the visibility of the Standard Buttons that are available for all Views
    Editor Buttons Controls the visibility of the Editor Buttons at the top of the Editing pane.
    Fragment Buttons Controls the visibility of the Fragment toolbar, to the right of the Editing pane. If the current document has a Type with Fragments defined, and some of the fragments have associated icons, these icons will be displayed vertically beside the Editing pane.
Editor Properties Controls the settings for some of the Editor's properties, including the margins, tag completion, syntax highlighting and soft wrapping. Having these items on the menus, rather than in the Preferences dialogs, facilitates the creation of keyboard shortcuts, improving usability.
Viewer Properties Controls the settings for some of the Viewer's properties, including the visibility of attributes, comments, processing instructions, etc.
Outliner Properties Controls the settings for some of the Outliner's properties, including the visibility of attribute values and text content, and the automatic creation of required nodes.
Grid Properties Controls the settings for some of the Grid properties, including how mixed content is handled.
Select Document... When there are multiple documents open, quickly move between them using the keyboard with Alt+1 or using this menu item.
Split Horizontally... When there are two or more documents open, display them one on top of the other for concurrent viewing/editing.
Split Vertically... When there are two or more documents open, display them side-by-side for concurrent viewing/editing.
Unsplit Undo the split that caused the current document to appear in a new window.
Synchronise Splits on XPath When two or more versions of the same document have been split into different windows, synchronise the scrolling of the multiple windows based on the XPath of the current element.
Full Screen Maximize the screen real estate available for the current document by hiding the Output and Controller windows. Repeating this function will cause the two hidden windows will reappear.
