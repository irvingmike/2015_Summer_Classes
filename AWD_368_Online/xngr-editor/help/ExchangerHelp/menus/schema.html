Schema Menu

Validate XML Schema If the active document is an XML Schema, validates it and reports any errors in the message window at the bottom of the application.
Validate DTD If the active document is a DTD, validates it and reports any errors in the message window at the bottom of the application.
Validate Relax NG If the active document is a Relax NG Schema, validates it and reports any errors in the message window at the bottom of the application.
Create Type Supports the editing of XML files that may reference unavailable schemas or to map DTDs or RelaxNG schemas to the corresponding XML Schemas so as to take advantage of the built-in Schema Viewer and Outliner and also functionality such as context-sensitive tag prompting.
Set Type... Associate the active document with a pre-defined Type, thereby facilitating context-sensitive tag prompting, browsing the Schema and editing in the Outliner.
Type Properties Displays the settings for the Type associated with the current document, and allows modifications.
Manage Types Allows the user to create, update and delete Types.
Infer Schema Create an XML Schema from a sample XML instance.
Convert Schema Allows the user to convert a DTD or Relax NG schema to XML Schema.
Schema Instance Generation Allows the user to generate a sample document from an XML Schema.
