Tools Menu

Show SVG If the active document is an SVG file, displays it using a built-in viewer.
Convert SVG If the active document is an SVG file, allows the user to convert it to JPEG, PNG or TIFF.
Send SOAP Message If the active document is an SOAP message, sends it to the service and displays the returned data in a new document or in the message window at the bottom of the application.
Analyse WSDL A tool for analysing a Web Services Description Language (WSDL) document and generating appropriate SOAP requests for invoking operations described in the document.
Clean Up HTML Converts arbitary HTML content to XML.
XML Diff and Merge A tool for performing an Unordered XML Difference between two files. Also includes a Merge facility for reconciling differences.
Start Browser Opens the active XML document in the user's default internet browser.
